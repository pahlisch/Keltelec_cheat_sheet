Tonality
Tonalité
Select tonality
Choisir la tonalité
[8] ♪ → hold 4
Notes cycle: Bb low → C → D → F → G → A → Bb high → repeat. Release on the note you want.
Les notes défilent : Sib grave → Do → Ré → Fa → Sol → La → Sib aigu → boucle. Relâcher sur la note voulue.
All settings (pitch, volume, scale, pressure) are saved per tonality.
Tous les réglages (pitch, volume, gamme, pression) sont sauvegardés par tonalité.

Octave mode
Mode d'octave
Select mode
Choisir le mode
[8] ♪ → hold 3
Two notes alternate while you hold the contact:
Deux notes alternent tant que vous maintenez le contact :
Release on low note Relâcher sur note grave
→ OCT covered = octave up
→ OCT bouché = octave sup.
(touch OCT hole to go up)
(toucher OCT pour monter)
Release on high note Relâcher sur note aiguë
→ OCT uncovered = octave up
→ OCT débouché = octave sup.
(slide thumb off OCT to go up)
(glisser le pouce hors d'OCT pour monter)
This setting is global (same for all tonalities). With bagpipe adapter, OCT is ignored — bag pressure controls octave instead.
Ce réglage est global (commun à toutes les tonalités). Avec l'adaptateur cornemuse, OCT est inopérant — c'est la pression de poche qui gère l'octave.

Toucher 1 et 7 simultanément. 10 notes brèves confirment. Il faut éteindre et rallumer avant un nouveau SOS.
